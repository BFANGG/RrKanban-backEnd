## RrKanban-backEnd
Trello-like web application backEnd 看板风格的项目管理工具后端
### tech
- **Node.js (v16.0.0)**：服务器运行环境。
- Express：Web 应用框架。
- Sequelize：ORM 工具，简化数据库操作。
- 认证：
    - bcrypt：密码加密。
    - express-session：会话管理。
    - passport & passport-local：用户认证。
    - Unsplash API：提供图片资源。
